Getting to know Cadenza

In order to begin creating your masterpiece, you'll need to become familiar with some how Cadenza creates music.


Powered by the VexFlow music notation language, Cadenze takes input from users that is written in VexTab. Below are some of the key commands you'll need to get started:

"stave" - Generate a new staff.

"tabstave" - Generate a tabstave (double staff); note that you can add as many of these as you'd like!

"voice" - Creates a staff for voice compositions.

"notation" - Set this equal to true AFTER you've created however many staffs using the above commands.


Creating notes:

Creating notes in Cadenza is a very simple process. Use the following command format as an example:

notes D-E-G/4

To add sharps, naturals, and flats to your composition, the previous command can be modified into the following:

notes D#-En-G@/4


A list of other useful commands to know to fully utilize Cadenza:

"clef" - Set equal to treble, bass, alto, tenor, etc.
