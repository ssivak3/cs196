Getting to know Cadenza:

In order to begin creating your masterpiece, you'll need to become familiar with some how Cadenza creates music.


Powered by the VexFlow music notation language, Cadenze takes input from users that is written in VexTab. Below are some of the key commands you'll need to get started:

"stave" - Generate a new staff, and you can create as many of these as you'd like.

"tabstave" - Generate a tabstave (double staff); typically used for guitar compositions.

"voice" - Creates a staff for voice compositions.

"notation" - Set this equal to true AFTER you've created however many staffs using the above commands.


Creating notes:

Creating notes in Cadenza is a very simple process. Use the following command format as an example:

notes D-E-G/4

To add sharps, naturals, and flats to your composition, the previous command can be modified into the following:

notes D#-En-G@/4

To add rests, insert two hashes (##) after the set of notes you've created. For example:

notes D-E-F/4 ##


Expanding your compositions:

To create a new measure in your composition, insert " | " after the first set of notes you've created. For example:

notes D-E-F-G/4 | D-E-F/4 ##


A list of other useful commands to know to fully utilize Cadenza:

"clef" - Set equal to treble, bass, alto, tenor, etc.

"tablature" - Set equal to true, or don't declare at all if you'd like the tabstave to be present beneath your notation stave, and false if you wish to remove it.

:8 - Place before your string of notes to create eighth notes. See the following for an example:

notes :8 D-E-F/4

:16 - Place before your string of notes to create sixteenth notes.

:32 - Place before your string of notes to create thirty-second notes.
